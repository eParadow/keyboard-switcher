Keyboard Switcher

Passe automatiquement d’une disposition clavier à l’autre selon l’application au premier plan (par ex. AZERTY au bureau, QWERTY pour certains jeux).

Télécharger

Installeur NSIS 64 bits. Après installation : icône dans la zone de notification, raccourcis bureau et menu Démarrer.

Télécharger l’installeur (.exe) Toutes les versions

Si le lien direct ne fonctionne pas encore, ouvrez la page des releases et téléchargez le fichier Keyboard-Switcher-Setup-x64.exe.

Fonctionnalités

  • Règles par processus — associez un exécutable à une disposition (HKL).
  • Interface de configuration — liste des processus actifs, pause du basculement auto.
  • Démarrage avec Windows — option d’auto-lancement.
  • Respect du choix manuel — si vous changez de layout à la main, l’app s’aligne sur votre préférence pour ce processus.
